❓ FAQ

口ゴボ矯正、
気になること全部答えます。

Q 口ゴボはマウスピース矯正で治りますか？ +
多くのケースで改善が可能です。当院はマウスピース矯正の症例数で日本No.1※の実績があり、口ゴボの治療を得意としています。まずは無料検査で、対応可能かどうかを確認いたします。
※クリアコレクト症例数
Q 自分の口ゴボがどの程度改善できるか事前にわかりますか？ +
はい。精密検査後、3Dシミュレーションで治療後の歯並び・横顔のラインをお見せできます。「ゴール」が見えた上で治療を始められるので安心です。
Q 痛みはありますか？ +
マウスピース矯正の痛みは、ワイヤー矯正の約1/10程度です。装着初期に軽い違和感がある場合がありますが、1週間ほどで慣れる方がほとんどです。
Q 治療期間はどのくらいですか？ +
口ゴボの程度や歯並びの状態によりますが、1年〜1年半が目安です。光加速装置を併用することで、約6ヶ月短縮できる場合もあります。※個人差があります。
Q 矯正中、周りにバレませんか？ +
透明なマウスピースなので、ほとんど気づかれません。接客業や営業職の方も多く治療されています。
Q 他院で「マウスピースでは難しい」と言われました。 +
ぜひセカンドオピニオンとしてご相談ください。当院では、他院で対応が難しいとされた口ゴボ症例の治療実績も多数あります。
Q 相談したら、治療しないといけませんか？ +
いいえ。相談・検査だけでも大歓迎です。無理に治療をおすすめすることはありません。検査結果を持ち帰って、ゆっくりご検討ください。
Q 分割払いはできますか？ +
はい。分割払いは2種類ご用意しています。
① 院内分割（無制限コースのみ／金利0％）：頭金20万円＋月々28,000円×12回
② デンタルローン（金利4.9%・最大84回）：月々約7,500円〜（コースにより異なります）
詳しくは無料相談時にご案内します。
Q 口ゴボの治療は医療費控除の対象になりますか？ +
噛み合わせの改善を目的とする矯正治療は、医療費控除の対象になります。確定申告で税金の一部が還付されるため、実質的な負担を軽減できます。詳しくはカウンセリング時にご説明します。
